# ИТОГОВАЯ АТТЕСТАЦИЯ
## МОЯ ПРОГРАММА
Random Quote Generator (Генератор случайных цитат)

Автор: Вероника Щербинина

Дата сдачи: 15.05.2026

## ОПИСАНИЕ ПРОГРАММЫ

Генератор случайных цитат с возможностью:
- Получать случайную цитату
- Добавлять свои цитаты
- Смотреть историю
- Фильтровать по автору и теме
- Сохранять данные в JSON

## ТРЕБОВАНИЯ ДЛЯ ЗАПУСКА

Python 3.6+
Tkinter (встроен в Python)

## КАК ИСПОЛЬЗОВАТЬ

1. ЗАПУСК:
   python quote_generator.py

2. ПОЛУЧИТЬ ЦИТАТУ:
   Нажмите кнопку "Сгенерировать"

3. ДОБАВИТЬ ЦИТАТУ:
   - Введите текст
   - Введите автора
   - Выберите тему
   - Нажмите "Добавить"

4. ФИЛЬТР:
   - Выберите автора или тему в списке
   - Выберите пустую строку для сброса

## ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
### Пример 1: Получение цитаты
Нажать "Сгенерировать" -> 
Результат: "Будь изменением... — Ганди"

### Пример 2: Добавление цитаты
Текст: "Код — это поэзия"
Автор: "Аноним"
Тема: "Программирование"
Нажать "Добавить" -> Цитата сохранена

### Пример 3: Фильтр
Выбрать автора "Джобс" -> 
В истории только цитаты Джобса

### Пример 4: Ошибка при добавлении
Оставить текст пустым -> 
Ошибка "Заполните текст и автора!"

## ТЕСТЫ
1. Генерация без цитат -> ошибка
2. Добавление пустого текста -> ошибка
3. Добавление без автора -> ошибка
4. 5 генераций подряд -> 5 записей в истории
5. Фильтр по автору -> только его цитаты
